# 安装 Nginx

## linux 安装 nginx 通用脚本

使用方法：

```sh
wget --no-check-certificate --no-cookies https://raw.githubusercontent.com/dunwu/linux/master/codes/deploy/tool/nginx/install-nginx.sh
chmod -R 777 install-nginx.sh
./install-nginx.sh
```

脚本会下载解压 nginx 到 `/opt/nginx` 路径下。

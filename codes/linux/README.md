# 环境部署工具

> **本项目脚本代码用于在 [CentOS](https://www.centos.org/) 机器上安装常用命令工具或开发软件。**
>
> 我选择了本人在日常开发中会用到的工具或软件。（不定期更新）

作为一名后端服务器开发，工作中，难免遇到拓荒（在新机器上部署应用）。

重复劳动总是让人厌倦，为此我自己写了一套脚本代码，用于环境部署。

## 说明

下载本目录所有文件到您的 Centos 机器上。

执行以下命令，进入操作界面，然后根据提示选择性安装环境。

```sh
$ chmod -R 777 *
$ ./main.sh
```
